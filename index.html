Educational Code and Tools

This repository contains demo code and tools to be used for educational purposes.
The following table gives a non-comprehensive list of the repo content:

CodeStepByStep Collection of exercises proposed to be incorporated into CodeStepByStep practice website.
CompCode Competition problems from various sites like USACO and AlphaStar Academy.
GraphVisualizer Javascript application useful for designing, visualizing and interacting with graphs as abstract data structures (ref: live tool).
TestsManagement Java tool for generating randomized tests for AP CS-A and Data Structures courses.
web-apis Collection of web apis (java servlets) serving as backend for a variety of web apps (ref: index.jsp).
drawing-lib Class library offering a generic canvas for drawing operations in an awt frame. This is a Maven project, generating drawing-lib-*.jar and drawing-lib-*-shaded.jar packages. The first one includes just the library, while the second one includes all dependencies. The documentation is online at drawing-lib javadocs. Its usage is demonstrated in unit tests.
